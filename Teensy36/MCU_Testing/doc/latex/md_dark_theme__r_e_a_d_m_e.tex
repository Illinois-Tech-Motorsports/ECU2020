Usage is very simple. You just need to modify your doxygen input file


\begin{DoxyCode}{0}
\DoxyCodeLine{\# Add files to extra stylesheet}
\DoxyCodeLine{}
\DoxyCodeLine{HTML\_EXTRA\_STYLESHEET  = "{}custom.css"{} \(\backslash\)}
\DoxyCodeLine{                         "{}custom\_dark\_theme.css"{}}
\end{DoxyCode}


If you wish to use custom made H\+T\+ML header and footer, enable this feature with


\begin{DoxyCode}{0}
\DoxyCodeLine{\# Custom header}
\DoxyCodeLine{HTML\_HEADER            = "{}html\_header.html"{}}
\DoxyCodeLine{}
\DoxyCodeLine{\# Custom footer}
\DoxyCodeLine{HTML\_FOOTER            = "{}html\_footer.html"{}}
\end{DoxyCode}
