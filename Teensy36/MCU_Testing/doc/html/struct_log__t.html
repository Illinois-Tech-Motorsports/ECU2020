Detailed Description

Base class used to log things to serial
This class has the capability to modify all of it's functions to instead use integer IDs
Refer to the Pre_Build.py script for more information.

Note
If Pre_Build.py is run, All strings passed to parameter message must be an inline string
Log("String goes here"); ✔️
Log(someStringReference); ❌
